# ✅ Todo App

A simple and responsive **Todo Application** built with **React**.  
This app allows users to add, view, and manage daily tasks efficiently with a clean UI.


---

## ✨ Features
- Add new tasks
- Display a list of todos
- Mark tasks as completed
- Delete tasks
- Reusable React components
- Clean and responsive UI
- Tailwind CSS support

---

## 🛠️ Technologies Used
- React.js
- JavaScript (ES6)
- CSS / Tailwind CSS
- Create React App
- Git & GitHub
